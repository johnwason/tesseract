This is a descartes sampler for a robot + external positioner.

This assumes the tool_pose is relative to the positioner tip link.

Parameters
target_poseThe The target pose to sample
tool_pose_samplerThe target pose sampler function to be used
positioner_kinematicsThe forward kinematics object for the external positioner system
robot_kinematicsThe robot inverse kinematics object
collisionThe collision interface
curret_stateThe current state of the system
positioner_sample_resolutionThe positioner system sampling resolution
robot_tcpThe robot tcp to be used.
robot_reachThe reach of the robot. Used to filter positioner samples.
allow_collisionIf true and no valid solution was found it will return the best of the worst
is_validThis is a user defined function to filter out solution
