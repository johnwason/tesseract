Detailed Description

The ExtensionApproachGenerator class For a series of waypoints that progresses along either the positive or negative x-axis of the waypoints, this class generates an extension at the start of that path. A positive X-direction in the supplied vector will automatically be transformed to align with the direction the path will travel, reducing complexity of calling code for alternating paths.
