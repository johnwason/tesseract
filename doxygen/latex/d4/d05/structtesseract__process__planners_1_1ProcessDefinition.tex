\doxysubsection{Detailed Description}
A process definition.

This is not intended to support or handle all processes but should covers most of the common ones like sanding, painting, grinding, etc.

In a process, we assume they follow this pattern.
\begin{DoxyItemize}
\item The robot starts from a start position,
\item Moves to an approach position just above the object
\item Executes a process(sanding, painting, etc.)
\item Return to the start position
\end{DoxyItemize}

Given the process described the user is only required to define two objects. The start position of the robot and a vector of Process Segment Definitions.
