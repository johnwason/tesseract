\doxysubsection{Detailed Description}
Used to store information about how a given \mbox{\hyperlink{classtesseract_1_1Tesseract}{Tesseract}} was initialized. See the \mbox{\hyperlink{classtesseract_1_1Tesseract}{Tesseract}} init methods. 

Note\+: Recreating a \mbox{\hyperlink{classtesseract_1_1Tesseract}{Tesseract}} does not guarantee that it is identical to the \mbox{\hyperlink{classtesseract_1_1Tesseract}{Tesseract}} associate with this construction info since the kinmeatics managers could have changed and it does not include the environment command history. This will simply recreate the \mbox{\hyperlink{classtesseract_1_1Tesseract}{Tesseract}} as it was at construction. ~\newline
